## 🖥️ What this app does

ai-invoice-agent helps you manage invoices with a chat-style interface. You can use it to:

- Find invoice records
- Check invoice status
- Review invoice details
- Ask simple questions about invoice data
- Keep invoice work in one place

It uses AI to help you talk to the app in plain language. It also uses cloud tools, so it can work as a web app with a modern interface.

Type the invoice number or client name into the chat box. The app should return matching records.

### Check payment status

### Review invoice details

Use the app to keep your invoice checks in one place. This helps when you need to review many records in a short time.
